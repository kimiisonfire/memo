switch 문에서 같은 값을 표출하는 정수 조건문이 있으면 같이쓰면됨
		case 9 : case 10 : 
			System.out.println("A"); 
			break;
		default :
			System.out.println("당장 나가세요");
		}

0~9까지 출력하는 for문 :: java는 항상 <= ~ < 니까 10까지
		for(int i=0;i<10;i++){
			System.out.println(i);
		}

for문 돌리는 연산자 i가 for문 안에서 선언된경우, i는 for문 안에서만 사용하고 밖에서는 없어짐


ctrl+f11 하면 전부다실행되자나
근데 필요한 부분에 ctrl+shift+b 누르면 라인 옆에 line break 생김
여기서 끊어서 하나하나 확인해주는겨 디버깅할수있게
여기서 그냥 f11누르면 java 모드에서 디버깅모드로 전환됨!

초록색 블록으로 디버깅할 포인트가 블록되어있는데 다음줄로 넘기고싶으면 f6
for문에서 f6하면 조건식-수행-증감조건식-수행 이렇게 왔다갔다함
이중 for문에서 안쪽 영역을 넘어가면 위쪽으로 넘어감

다음 breakpoint로 넘어가는건 f8 (이중for문에서는 바깥쪽 for문으로 넘어가겠지)

디버그에서 자바로 넘어왔을때 코드 수정하고 실행하고 하려면 뭔 창이 뜰거임
하단 debug탭에서 중지시키고 실행하세용
